EXPLORING TAGS IN HTML



  1. Audio Tag : It is an inline element that is used to embed sound files into a web page.



  2. Anchor Tag : The "a" tag defines a hyperlink, which is used to link from one page to another.

    For Example: Click here to move to Home Page



  3. Abbreviation Tag : The "abbr" tag defines an abbreviation or an acronym.

    For Example: HTML is the standard markup language for creating Web pages



  4. Address Tag : The "address" tag defines the contact information for the author/owner of a document or an article.

    For Example:

    Written by U Mahima.
    508-SP40, Portofino
    Lavasa.



  5. Bold Tag : The "b" tag specifies bold text without any extra importance.

    For Example: This is BOLD text



  6. Blockquote Tag : The "blockquote" tag specifies a section that is quoted from another source.

    For Example:

    Wikipedia states that:

    The HyperText Markup Language or HTML is the standard markup language for documents designed to be displayed in a web browser. It can be assisted by technologies such as Cascading Style Sheets (CSS) and scripting languages such as JavaScript.



  7. Break Tag : The "br" tag inserts a single line break.

    For Example:
    Hi!
    This is U Mahima
    from 5BSCDS



  8. Button Tag : The "button" tag defines a clickable button.

    For Example:



  9. Delete Tag : The "del" tag defines text that has been deleted from a document. Browsers will usually strike a line through deleted text.

    For Example: I am from 4BSCDS 5BSCDS.



  10. Details Tag : The "Details" tag specifies additional details that the user can open and close on demand.

    For Example:

    My Info

    HI!
    This is U Mahima
    from 5BSCDS
    of Christ University, Lavasa



  11. Dialog Tag : The "dialog" tag defines a dialog box or subwindow. It makes it easy to create popup dialogs and modals on a web page.

    For Example:

    This is an open dialog window







  12. Footer Tag : The "footer" tag defines a footer for a document or section.

    For Example:

    Author: U Mahima
    u.mahima@science.christuniversity.in



  13. Heading Tags : The "h1-h6" tags are used to define HTML headings. h1 tag defines the most important heading. h6 tag defines the least important heading.

    For Example:

    This is heading 1

    This is heading 2

    This is heading 3

    This is heading 4

    This is heading 5
    This is heading 6



  14. Horizontal Rule Tag : The "hr" tag defines a thematic break in an HTML page. It is most often displayed as a horizontal rule that is used to separate content (or define a change) in an HTML page.

    For Example:

    HTML
    CSS



  15. Italic Tag : The "i" tag defines a part of text in an alternate voice or mood. The content inside is typically displayed in italic.

    For Example: This is text ITALIC style



  16. Image Tag : The "img" tag is used to embed an image in an HTML page

    For Example:



  17. Form Tag : The "form" tag is used to create an HTML form for user input.

  18. Label Tag : The "label" tag defines a label for several elements.

  19. Input Tag : The "input" tag specifies an input field where the user can enter data.


    For Example:







  20. Ordered List Tag : The "ol" tag defines an ordered list. An ordered list can be numerical or alphabetical.

    For Example:

    1. HTML
    2. CSS
    3. JavaScript


  21. List Tag : The "li" tag defines a list item. This tag is used inside ordered lists and unordered lists


  22. UnOrdered List Tag : The "ul" tag defines an unordered (bulleted) list.

    For Example:

    • HTML
    • CSS
    • JavaScript



  23. Paragraph Tag : The "p" tag defines a paragraph.

    For Example:

    Hi! This is U Mahima from 5BSCDS og Christ University, Lavasa.



  24. Table Tag : The "table" tag defines an HTML table.

  25. Table Row Tag : The "tr" tag defines a row in an HTML table.

  26. Table Heading Tag : The "th" tag defines a header cell in an HTML table.

  27. Table Data Tag : The "td" tag defines a standard data cell in an HTML table.


    For Example:

    Subject  Marks
    Acturial Science     83
    Introductory Economics     95




  28. Subscript Tag : The "sub" tag defines subscript text. Subscript text appears half a character below the normal line, and is sometimes rendered in a smaller font.

    For Example: H2O, O2



  29. Superscipt Tag : The "sup" tag defines superscript text. Superscript text appears half a character above the normal line, and is sometimes rendered in a smaller font.

    For Example: 32, a4
